### 해결한 이슈들

1. **CORS 에러** → `app.enableCors()` 추가 + 프론트엔드를 Next.js 프록시(`/api` 상대경로)로 전환
2. **API 라우팅 404** → `.env.local`의 `NEXT_PUBLIC_API_URL`을 `/api`로 변경, `next.config.ts` rewrite 설정 완료
3. **회원가입 500 에러** → `SignupUserDto` 필드 매핑 수정 (firstName, lastName, majorId), ValidationPipe 활성화, majorId 드롭다운
4. **로그인 401 에러** → `access_token` → `accessToken` 프로퍼티명 수정 (login/page.tsx + auth.ts 양쪽)
5. **로그인 2번 해야 되는 문제** → `router.push` → `window.location.href`로 전체 페이지 리로드
6. **대시보드 Hydration 에러** → `mounted` 상태를 추가하여 SSR/CSR 동일 렌더링
7. **시간표 생성 404** → `getCurrentSemester()`가 2026 SPRING 학기를 못 찾아서 발생. 없으면 자동 생성하도록 수정
8. **시간표 상세 TypeError** → `semester: true` include 추가 + 프론트 optional chaining
